# vue-template

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

```
yarn build
```

### Run your tests

```
yarn test

```
yarn lint

```
yarn fix
```

### Run your end-to-end tests

```
yarn test:e2e
```

### Run your unit tests

```
yarn test:unit
```
